Application description

Simple web application for data visualization. It expect backend to be running on http://localhost:8080.

Technologies used

  • React

  • Redux + redux-thunk (for async API calls)

  • Bootstrap (Gentelella admin template)

  • JQVectormap

  • ChartJS

  • Webpack

  • Node.js

Build process

This project can be build both for development and production use. In development mode webpack will run dev server on http://localhost:8088. In production mode resulting files can be found in /dist folder.

Development mode:

  1. Navigate to same directory where package.json is located.

  2. Run npm install. This downloads all necessary dependencies.

  3. Run npm run dev. This runs development server on http://localhost:8088.

Production mode:

  1. (Optional) If you need to change host API, you can edit /src/apiConfig.js

  2. Navigate to same directory where package.json is located.

  3. Run npm install. This downloads all necessary dependencies.

  4. Run npm run production. This compiles files for production use.

  5. Files from /dist directory can be placed into public web folder for any web server.
